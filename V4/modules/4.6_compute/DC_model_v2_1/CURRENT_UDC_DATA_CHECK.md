# V1.1输入接口状态

本版本不附带用户数据，运行时按以下规则读取：

1. 老师168小时任务曲线作为相对时序形状，自动映射到 `simulation_start`。
2. Copernicus 21天NetCDF自动截取仿真周，选择约50 m的最近有效海洋格点并插值到小时。
3. 基准场景采用常量电价372.70元/MWh和8.20 MW DC功率上限，不读取24小时电力接口。
4. 经济计价使用元/GPU·h；通过 `equivalent_gpu_it_power_kw` 将MWh-IT转换为GPU·h。
5. ERA5、风力数据不进入DC-only模型。

本文算力服务价格参考AutoDL算力云公开的GPU按量租赁价格。该平台当前普通用户价格约为：H800 8.88元/GPU·h、H20 7.58元/GPU·h、PRO 6000 5.98元/GPU·h、A800 4.98元/GPU·h、L20 3.68元/GPU·h、RTX 5090 2.88元/GPU·h和V100 2.28元/GPU·h。考虑不同任务对算力性能、连续占用和服务保障水平的要求，本文分别设置刚性推理、柔性批处理、连续训练和可选弹性任务价格为8.0、4.0、6.0和2.5元/GPU·h。上述价格属于基于公开市场挂牌价构造的情景参数，并非运营商针对不同任务类型发布的直接报价

经独立测试：168小时优化可正常求解，动态PUE随负载和海温变化，任务平衡、队列守恒和功率上限审计通过。
