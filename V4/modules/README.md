# V4专业模块目录

本目录按4.1/4.2冻结责任划分，不按原工程的历史文件夹命名。每个模块包含原模型或必要参考代码，以及V4专属`integration/`接口层。

```text
4.3_source/             去重后的多源供给正式模型、验收、文档和源侧适配器
4.4_bus/                公共母线唯一总账及历史参考核心
4.5_storage_hydrogen/   电池、制氢、储氢及状态提交
4.6_compute/            DC-only算力模型、必要数据和接口桥
4.7_outputs/            电力/氢/算力/海洋交付通道
4.8_objectives/         经济/环保/可靠性评价与KPI
4.9_dispatch/           跨模块调度请求与约束校验
```

联合运行只从各模块`integration/`和明确列出的运行核心进入。`reference_model/`用于追溯原方程，不进入公共总账。
